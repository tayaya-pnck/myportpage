Lung Sound Classification Based on Audio

Built a lung sound classification system using respiratory audio, extracting acoustic features via convolutional neural networks (CNN) and modeling temporal dependencies with Bidirectional GRU (BiGRU). The model effectively distinguishes inhalation, exhalation, and adventitious sounds (e.g., crackles, wheezes), improving upon baseline CNN-only approaches.

Key Features:

  • Classifies lung disease based on respiratory audio
  • Removes external noise that affect lung sound using adaptive denoising pipeline
  • Comparing 2 different models (CNN-BiGRU and CNN-LSTM) with both denoised and original data
